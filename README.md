# Проект Mesto

Учебный проект для [**Яндекс.Практикум**](https://praktikum.yandex.ru/), посвященный отработке навыков JS и адаптивной верстке.
Сайт доступен по ссылке: https://tonilyn.github.io/mesto/

## Интро

Проект сверстан по данному [макету](https://www.figma.com/file/StZjf8HnoeLdiXS7dYrLAh/JavaScript.-Sprint-4) в **Figma**.

## Используемые технологии

* JavaScript, HTML, CSS
* Figma
* Flex-верстка
* Grid-layout
* Сглаживание шрифтов
* Слезы, смех и пять пачек кофе

## Изображения
Изображения взяты из [Unsplash](https://unsplash.com/). Все парва принадлежат авторам.
